2.通过BIOS进入电脑设置U盘第一启动

联想电脑：

进入PE后，必需先将设置好的u盘插入电脑，开机时候按F1进BIOS设置或是用快捷键设置，在开机的时候不停的按F12就会弹出开机启动项设置选框，选择usb。[8]

如果你的电脑可以直接进入登陆界面

要求输入密码时，按两次Ctrl+Alt+Delete，跳出帐号窗口，用户名输入：administrator，回车。

如果你可以进入带有命令提示符的安全模式

启动电脑/系统的时候不停按F8，进入Windows启动选项：选择带命令符的安全模式[9]。

此处有两种可能：

1:直接需要输入密码界面。

如果是此状况，引用知乎用户鲤鱼霜[10]的回答，当然这也仅仅是运气：

这一步我的电脑不知咋回事选了带命令符的安全模式后弹不出command prompt窗口，而是直接到让输入密码的界面，在我不停地试密码无果后电脑不知咋滴就蹦出来了cmd窗口。

之后便到了第二种，即直接跳出命令提示符（CMD窗口）——我也很希望你的电脑可以直接弹出此窗口。

根据这个链接-rzhy的回答，可能是说“网上说的那些都是指administrator没有密码的情况下”。

2:Administrator跳出Command Promp

使用密码错误的CMD窗口添加增加管理员权限的账户

弹出cmd窗口后的提权通用有效过程

  1. 添加新用户：输入:

    net user admin /add

    回车。

    含义：加一个名为”你自己设定的用户名“到user组（用户组）

    这里的admin，可以随便更改成别的你想设置的用户名。

  2. 让新用户成为管理员：cmd下输入：

    net localgroup Administrators admin /add

    将我们建立的admin用户提升管理员，将你自己设定的用户名的用户权限升到administrators级别

  3. 回到cmd下，输入：

    net user admin /active:yes

    激活新账号

  4. 重启电脑,选择新建的账号进行登录。

之后的可选项：控制面板-用户帐户-忘记密码的用户-删除密码
