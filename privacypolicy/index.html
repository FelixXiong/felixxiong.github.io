隐私政策

更新日期：2026年5月24日

引言

DayTrace（以下简称"本应用"）由开发者个人开发并运营。我们非常重视您的隐私保护，本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。请您在使用本应用前仔细阅读并理解本隐私政策的全部内容。

一、我们收集的信息

本应用采用本地存储架构，您的所有数据均存储在设备本地，具体包括： 1. 日记内容：您撰写的日记文本及关联的情绪标记 2. 事件记录：您创建的事件名称、进展描述 3. 消费记录：您记录的消费金额、类别及备注 4. 应用设置：您的功能偏好设置（如闪控球开关状态） 本应用不会收集您的设备信息、位置信息、通讯录或其他与应用功能无关的个人信息。

二、信息使用目的

我们收集的信息仅用于以下目的： 1. 为您提供日记记录、事件跟踪和消费管理功能 2. 在应用内展示您的历史记录和统计数据 3. 支持数据的本地备份与恢复

三、数据存储与安全

1. 本地存储：所有用户数据均使用 HarmonyOS Preferences API 存储在设备应用沙箱内，不会上传至任何服务器 2. 数据隔离：应用数据受到系统沙箱保护，其他应用无法访问 3. 备份文件：当您使用导出功能时，备份文件将保存到您指定的位置，请妥善保管

四、信息共享与披露

本应用不会与任何第三方共享、出售或披露您的个人信息。本应用不包含任何第三方 SDK、广告组件或数据分析服务。

五、您的权利

您对您的个人信息享有以下权利： 1. 访问权：您可以随时查看应用内存储的所有数据 2. 修改权：您可以随时编辑或更新您的记录 3. 删除权：您可以删除单条记录或通过"清空所有数据"功能删除全部数据 4. 导出权：您可以随时通过备份功能导出您的所有数据

六、未成年人保护

本应用适合所有年龄段用户使用。如果您是未满14周岁的未成年人，请在监护人指导下使用本应用。

七、隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在应用内公布，建议您定期查阅。如果更新涉及您的重大权益变更，我们会通过应用内通知的方式告知您。
