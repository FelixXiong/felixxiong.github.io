在同一天空下，做着畸形的产品

发表于 2019-09-28 | |
字数统计: 2.2k | 阅读时长 ≈ 7

按照常理来说这篇文章应该是在微信公众号产品北里上写的，因一些风险需规避，故此出版在此发布。

微信公众号白天或近期可能会有一个“和谐版”，敬请期待。

安全声明：博客仅写给自己看，请勿联想与传播，否则将追究相应责任。

写在国庆节前。

互联网人对现实事物的批判，可能也只能从工作-产品角度进行了。

需求南口
关键词：政策 | 需求

作为产品经理看到这件事，能做的就是迅速截图，之后可以做也是唯一能从产品的角度，对这件事情进行批判，以及更多对“需求”的定义、“进入一个新的行业”的第一个产品、“资本的力量”的思考以及反思——

必须得有人站出来说话。

以下几个现象，都是在国内大环境下我们“心知肚明”的事情，通常都是“只可意会，不能言传”的事情，这里使用了几个排比句，从今天看到的一条动态说起。

当我们一出门就是探头时

随着人工智能的日益成熟以及大数据的杀熟，我们几乎每天都可以收到各式各样的垃圾营销短信，并不知道ROM的背后，是否仍旧在以“用户体验计划”的名义上传用户的数据，监控着用户的广告ID。

在这里推荐一个回形针的视频：摄像头如何监控十三亿人？

但是有这些，又有什么用呢，真正出了事，硬盘还是会坏的啊。

这是自己拍的一些照片。

当我们看到孩子成为受管制的物品时

有需求就有市场。

这是在脉脉上看到的一条动态。

这一次，自己可能是真的怒了，直接在朋友圈中发评论道：

我恨侵犯人权的企业。

脉脉里有人说这是中国教育的悲哀，甚至我们一出门就被摄像头监视着。

也有人说，如果旷视不去做，也会有下一个旷视去做。

找不到原片，只能用自己的截图代替了。

他们，终于要对孩子下手了。监控孩子在上课时的听讲、举手、写字的次数，估计是不敢根据评论区的进行认领产品，即使认领了可能也会被现实中的孩子（受害者）和家长（重视孩子隐私的明智的视野开阔的家长）乱棍打出，这片土壤总能让这些“奇怪的”“看似很不合理的”东西变得合理化，这就是一种畸形之处。

其他的畸形，会在下文中提到。

非常遗憾的是，我们没有相应的保护隐私的自由组织，没有途径去合理的反映我们的诉求，及时反映了，也起不了任何作用。

当我们的反馈起不到任何作用时

这里的反馈的产品指的是（大多是）亿活用户的产品

腾讯的外部反馈，仅仅是局限于“吐个槽”产品、部分少有的在线客服，内部的反馈却很迅速，以及那些“优秀的”、“固执己见“并完全不care他人的想法时，自己做着固执的产品。

而之前在饭否上，张小龙曾说：

互联网产品应该是由用户推动，而不是产品经理来推动。产品经理的作用只是找到四两拨千斤的地方稍微用点力。这方面，饭否给了个表率。这么久以来，他们只是添加了“测试版”三个字，其它什么都没变。对此，我很感动。

—— By 2010-12-25 18:14

如果是普通人或者是新晋产品，就会被领导和同事教导：

你要学以致用。

可是张小龙呢？感动过了也就是感动过了而已，什么也不做。

结果，自己被打脸。

这一切到最后，即使我发现这是在给“新晋”产品经理起反例：给我们做了一种反面的不听用户反馈的总会有一众产品经理去帮“神级‘产品经理佐证：

一个上亿DAU的产品，他的每一次变革都会有蝴蝶效应。

人无完人吧，很难去评价他。

这些小缺陷不能密盖他们的成功。

或者自己宣称：

同行 with us。

2019年微信公开课，来源：2019微信公开课张小龙演讲全程回顾，总结微信8年

但你为什么要做的这么闭环呢？

竞争对手在你的应用里分享，你为什么要屏蔽别人呢？因为抢饭吃么？

或者用罗永浩的一句话讲，企业家不怕被打脸。

总之一句话，爱用不用。

成功者，做什么都对？

还有马云说的一句话，一众国内媒体都争相报道，例如前段时间的WAIC，前面还说“996”是年轻人的福报，后面即将卸任之时说，应该每天工作三天，每天工作四小时。

对于媒体来说，为了夺眼球，在这个畸形的社会环境下生存下去时，什么都可以做得出来，什么都可以被改变。

当我们无法修改头像

系统升级，系统维护，系统异常

上面所列举的词，是国内环境下的人都懂这个词的含义。

每到一些特殊时期，当然，每天都有可能成为那一天。

百度云系统升级，微信微博无法修改头像，提示系统异常，在国内大环境中这些看似“正常”的事也慢慢的由于我们的习以为常而被淡忘。

即使如此，我们都心知肚明，但就没有人去说，即使说了，也会顿时消失在庞大的国内网络里，当然，很多人也不敢说。

于是，后来者必须知道这是一条国内产品的 “潜规则”，而自发的让人感到迷惑不解：为什么这个产品又挂了，是不是又侵权了，但只要联想到实际生活中，这一切可能就很明朗了，但这一切其实都不是这么的明朗——可能还有你不知道的事情，也没有人会去告诉你的事情。

正如我写的微信开头所说的那样，

喊不动白底黑字。

发问

遇到这些问题，自己总有一点好奇，这些CEO们这些产品经理们，为什么都没有骨气不去做这些事情？是因为大企业很难进，面试需要好几轮，而为了周围人保持的面子继续做着这些违心的事情？

影响了用户体验甚至是违背，那么多次，这算是一个合格的产品经理么？

非常简单的例子：降热搜。

或者是不愿放弃自己一手因为艰难险阻而创立的从0-1开始的企业？

不过，即使你不做，也会随便找个人来顶替你这个当前的职位，并且随便找个原因就可以把你给辞了。

可能因为吃饭不易吧。

例如之前有一段时间，截图式听歌看评论。

说完了这些，在阅读此篇文章的你，可能会觉得这也许都是深夜里的矫情。

但我说，我们在这畸形的社会里总得有人站出来发声、说话，哪怕又会瞬间淹没在有形的但看不见的特殊网络里。
