22岁那年，我掉入了手机坑

发表于 2019-11-02 | 更新于: 2019-11-17 | 分类于 生活 ， 精神日志 | | 阅读次数：
字数统计: 2.3k | 阅读时长 ≈ 7

原文名称：别人在发展的路上走了又走，我在换机的型号上愁了又愁

关于手机的折腾，从我的第一部智能手机华为C8812开始在手机上耗费的时间颇多：不断对手机进行刷机，不断模仿着iOS（记忆深刻的就是当时还是拟物化的系统），直到影视动画专业的中专，开始绘制原型图，再到今年十一月购买的坚果Pro3，再到撰写本文的时期。

时间的耗费和金钱的亏损，似乎真的是无法挽救的——还有对兴趣的极度痴迷，以及精神上的持续性寂寞以及无奈。

一

从小米MIX3 ，到坚果Pro3

刷着微博，看着何同学制作的摄影视频转发到了六万，钟颖大哥在微博转发的入职与离职Apple……

我却在坚果手机是否要下单、是否要购买白色以及第二天的摄影、出售闲置手机以及出售的渠道愁了又愁。

况且不说是因为资金原因，当初在广州看了两个月的MIX3。

于是我发自内心的感叹道：

有钱真好。

关于黑与白的论调，现在在酷安上已经有着不胜枚举的“统一”：白色是坚果以及锤子的经典款。

从之前的发布会觉得，前面板并没有白色所带来的“熊猫机”的感觉，再到看到京东1月2日零点截止的200元优惠券，事实证明我对新鲜事物的体验与渴望以及冲动，起码在用户心理学上还是对我很有作用的。

毕竟，我又亏了一千多，累计又亏了几千多。

手里的MIX3讲起来是自己购买，其实还是自己家人支持，说好的用一年，没想到最近总有出售的打算，可能还是因为刚开机的时候，前任公司的设计碰过以后，我便在V站上发帖：

于是同样的，在别人发展的时段，我左琢磨又琢磨，最终还是刷到了开发版，开始了漫长的“为MIUI提供我自己的ROM建议”之旅。

这段时间，我当时将小米社区放在桌面上，不断地寻找着小米的工作人员进行小米社区建议的提供，不禁想起之前在朋友圈对于小米社区“审核过于严格”的论调，在工作的时段看到MIUI11可以下载的时候，当时自己还在工作但确实没来得及进行升级，之后回家导致了被“撤包”风波…

此处应该会有一些图片，包括在朋友圈的吐槽、在脉脉的职言中表达了我对这样的审核机制的想法。

MIUI11的字重调节，地震速报（即使目前和身在江苏的我并没有什么关联）,全局便签（也是借鉴了闪念胶囊）…

这使得喜欢体验UI以及OS的我，在做产品经理的时候以及阶段，就已经明确了自己的职业方向–我自己想要什么：那就是：做UI或者OS的C端产品经理。

如果说我是如何使用我的MIX3的，也可以写出一大堆文字，即使我知道同时发布于酷安的看看号上、微信公众号上的话又可以收获一些干货以及阅读量以及粉丝，就先不在这篇文章里做赘述了。如果按我之前的逻辑，毕竟会透露点什么，但现在我不了。

在这个“以赚快钱为荣”的时代，并不能这么做，这就和我之前提到的文章里的一些论调是一样的。

随着坚果的发布，正好入了职，等着同事下班一起使用TNT以及产品的讨论，于是开始了文章的题头：别人在发展的路上走了又走，我在更换手机的型号上愁了又愁。

  1. 到底是R1，还是新的坚果手机。如果说是新的坚果手机，我有以下几点不满意（相较于小米MIX3）：

  2. 没有20W以上的快充

  3. 没有无线充电

  4. 没有NFC

  5. 水滴屏而不是全面屏

  6. 最后一点，白色还是熊猫机。

  7. 最终，决定了购买黑色，但心里还期望着购买白色，即使需要加400元。

二

出手闲置的烦恼

过于喜欢电子设备，导致闲置最后太多。

总希望或者把他们推翻重来，这将会让我的公众号有着新的一篇文章：《给用户一次推翻重来的机会》。但不知何时，才可以可以进行撰写和发布。

出手闲置的渠道也有很多，也不赘述——但几乎都是亏本买卖。

由更换手机引发的思考：失去的时间以及其他

更多的时候自己当然会想，我真的将手机用到了极致，不断地在花钱，那我也没有赚钱呢？

如果将这个时间花在一个地方进行深耕，那将会也是很棒的。

于是从2018年开始，我接触Swift，那是我正好在用iPhone6s，在手机上试运行了我的第一个开源项目《孤岛》：

最终毕业后没能继续深耕Swift，后面出现了Swift UI。烂尾。

在南京的时候有一次千载难逢的机会我收到了锤子科技人事的电话，但可想而知的结果，非常遗憾。

之后，带着锤子，北上广漂，一个人漂，只想着找工作，赚钱。

再之后，我接触了Android。

去广州面试14家Android，没能交得起房租而引发的焦虑，到现在做的产品运营，在编程学习上仍然是自己的非常短的一块短板，和沟通一样。

目的，或许仅仅只是为了做自己的产品，在不断的焦虑中修正自己的时间，让自己所花的这些钱进行高效的赎回，提升自己在技术上的支撑。

回顾一下自己学习Android以及Java的时间，起码也有一年。

于是，这像是一个循环，不过这个循环，就从发布闲置这里，断掉了，裂开了，洞口很大。

其实我我安全可以进行整理一个我自学的时间轴了。

明天收到坚果Pro3的黑色，原本还打算抢白色的，UFS2.1用起来也没什么感觉，又非旗舰，黑色就黑色，也无妨。

我说服自己道。

很荣幸，我结识到了一位也用锤子的同事，让独自发现产品的自己，不再孤单。

这可能就是所谓的“有趣的灵魂终将遇见”吧。

三

今年的双十一，我在傍晚进行下单

11月1日，我在不断的思维冲突下，最终在坚果R1和Pro3中进行抉择，下单了后者。

双十一的深夜，下单了Pro3的足迹背壳。先买了两个，后来又觉得发货慢，于是发起退货，浪费了10元的优惠券，在京东重新下单。

包括上班的时间内，等候着它的归来。

很多人说关于评测中拍照的只字不提，再到很多人期待一到手之后的One Step3.0也需要内测之后进行体验，之后用户们在坚果社区内大喊着“牛逼”。

最近的知识星球内，宣发了自己在办公室中的TNT体验，无限投屏。

结果依然是，没有一个评论。

四

似乎，还是需要通过R1进行过渡，Pro3并不合适。

渐渐发现，之前的一些需求全部需要使用R1进行弥补以及过渡，顺利从4G直到5G……似乎是我一个花钱买手机的一个通俗的借口。

如果这样，Pro3就会继续成为自己亏损的手机。

周末出门，去医院看焦虑症延期，去Apple Store，发现一些大二的幼教女同学男同学们，在学习Apple iWork系列，这些我早已熟练掌握。

上午，TNT的群里一个山东的朋友帮忙下了R1单，等我考虑是否入手。

五

22岁那年，我荒废的时间与纠结和无奈

这是在我的22岁，对于时间、电子设备以及时间、思维播散的焦虑，及对未来的恐惧。

这是在我的22岁，在小米、锤子的电子设备之间互相游走，同事就连写博客文章都发现越来越水，被生活推着走：

需要写文章了，发现需要的图片还没有做，

快要到网上的朋友帮忙下单的坚果R1的最后付款时间的一个小时了，发现自己依然在犹豫不决。

但是别人，都在学习，以及在看着书啊。
