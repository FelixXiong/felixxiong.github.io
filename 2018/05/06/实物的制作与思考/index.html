实物制作的与思考

发表于 2018-05-06 | 更新于: 2018-09-02 | 分类于 毕业设计（接本） | | 阅读次数:
字数统计: 1.3k | 阅读时长 ≈ 4

其实在设计的环节还没进行完时，就必须开始做实物了。，因为毕业展需要使用。

于是我想着可以产生的实物以及软件周边，到处询问。
当然，想法总是好的，只是……

从语音助手开始

首先尝试制作一个使用单片机进行振幅的装置。

我首先尝试联系商家，诉说着自己的需求，首先当然是文字。

能不能做

有单片机最小系统带有声音传感器、震动马达的装置，

玻璃里面注水，人一靠近声音越大振幅越大的装置？

淘宝的作用是新循序渐进的，他让我知道了其中是的装置：振动马达，单片机最小系统，声音传感器。

比较遗憾的是可能暂时找不到那家店铺了，后面再找吧。

淘宝使得一个人原本从不了解的领域进行最基础的科普，这也是百度贴吧作为“全球最大的中文社区”用户来说产品里面的一个增长的功能环节吧。

之后我便在贴吧里就进行询问。

关于价格上的坑

第二个，要人民币800.

根据我的回答来看，要750,理由是因为时间就是金钱，我这个比较复杂。

周围也没有懂单片机行业的同学以及朋友，所以这个也就不了了之了。成为了实物和想法之一。

这是我的思维导图,是用simpleMind Free制作的。

木质键盘

再次因为我制作毕业设计实物时有一个强大功能的自定义键盘，所以第一考虑到的就是木质的材料去制作。

因为考虑到木质材质制作工艺可能会简单一些，

首先想到的是和积木块一样的字母。后面我最后就说“就像积木一样”，最后的结论就是小点，只要精美就行,因为到了那时已经没有什么要求了，当然也不想去提出，部分原因是因为价格，更重要的是因为时间。

在这个过程中，经历了个次手绘草图。

关于一个要求点不能或者（看起来）很难满足的时候，就去写便签吧。

和上次制作打印的书籍一样，同样我也写了一个锤子便签进行需求上的说明和制作，之后直接发送这个便签就行。

再不行多几个手绘草图，就和上面一样。

其他周边

其他周边的话我选择了全部都为定制，以及更加实用的东西，例如手机壳，闹钟等。

帆布包

查找帆布包的时候，发现自己的欲望是会动摇的。

当然在时间非常的匆忙以及捉襟见肘的时候谁都会吧。

发现一个的过程中，就会去问，还有没有其他可以制作的。

闹钟

仅仅是一个添加了LOGO的闹钟，尝试将其中的反色LOGO横穿其中，其实因为淘宝商家的技术原因，无果。

在交流的时候的发现

在交流沟通过程中，有的商家很专业，例如如下对话：

上下灰边不要

同时也有的商家不是那么的专业。

需要考虑的点

  1. 对方厂家目前的订单也许已经排到很多以后
  2. 考虑现在制作，发货的时间
  3. 如果时间很赶，尽量在图已经制作好的情况下再去沟通
