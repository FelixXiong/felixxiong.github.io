朋友圈没有隐私--个人近况隐私从发布到被盗用以及盗用之后

发表于 2018-05-24 | | 阅读次数:
字数统计: 1.3k | 阅读时长 ≈ 4

当晚发的朋友圈，就被亲戚在未经允许的情况下盗图以及异常传播，我是怎么做的？
以下是我的行动以及反思。

我的原创文章图片原文。

也仅仅是在发给同学一个节日快乐发现已经是红色叹号的时候，再加上写了文章，就顺带回顾之前确实很难那一说出口的往事，写了很长的文章。

朋友圈写作背景

在写下这个文章的时候，都是最近的一些想说的话。以及和之前samer聊天的话。

写着写着在转换着same和微信的记录以及聊天窗口，回忆起最近的找工作等事情，却是感到委屈。
自己对自己的期待非常的大，行动力以及执行力缺失，
每天都很难开心，自己给着自己压力。
老师和同学，因为没有处的好，所以便有了离开。
由于之前一个关于大学生报名比赛的事情感到压抑（可能是文章没写完的原因）,
在进行发布的时候，关于一些家人看了也没用或者则觉得我是在抱怨的，就选择性的屏蔽。

在这个过程中我也遇到了很多人，从通讯录中选择了他们。毫不犹豫而又犹犹豫豫。
没想到这篇文章居然被一位亲戚盗用了，并且在未经允许的情况下。

和亲戚的沟通

多次尝试与亲戚沟通，侵犯了我的权益却需要我客气，并不知道是为什么。

于是有了被删微信好友的举动。

最后多次尝试以及帖子下方的回复，证明是锤子手机的原因。

关于微信的投诉和产品思考

以下是关于微信投诉好友的说明。

非好友关系

在用户使用朋友圈中，还有一种场景即和对方盗用朋友圈原创图片或者争吵之后，发诋毁对方的消息是很容易被删除好友的，不知腾讯是否考虑过。

此时是很难甚至是不可能添加微信的，所以朋友圈入口当然也无法进行查看，并且添加头像。

或许这方面，微信想的确实是过于简单了，仅仅是朋友圈盗图以及发送广告这么简单。

其他平台的链接汇总

知识星球-产品党星球上的链接：微信产品的朋友圈投诉功能确实是有问题。

锤子科技论坛的连接：[系统软件]锤子手机连微信的基本投诉功能都用不了。

为了突出自己的权益，自己已经使用iPhone6s进行举报，但都没有取得相应成效，不通过。

于是并未知晓对以后的生活的影响，是否涉及到人身安全。

关于公共关系

家庭与其他亲戚

家庭中父母对于此事的看法是：

发出来就是给别人看的，不如不发。

我的回应是：

我更宁愿去对网络上的陌生人去说……当时也考虑了，以为只会看不会转，所以可能就没有屏蔽她。

当然如果TA们（指的是网络上的陌生人）发朋友圈的时候屏蔽我……我也不会知道。

姑妈说：

是亲戚为什么要去举报？

我的回应：

原本就是我原创的，侵犯了我的权益啊。

姑妈说：

我如果这样周围的亲戚都会一个一个的失去，就像 QY（当时和她说关于设计一个欧普瓷砖的纹样，我提出要设计费，被得知之后即使有给但在家庭环境的压力下未及时收取，我觉得设计就应该被收费，即使是入门的设计）一样。

我的回应：

所处的环境就是这样，失去就失去吧。

同学

由于自己的疏漏（因为一些学长学姐并未添加“同学”的标签），或许也在同学中传开了。

发了此文之后于是联系同学（当时未添加标签的三位），问有没有看到我的朋友圈，说是没有，如果看到了也不会承认的。

自己后续一些关于博客的更新文章，同样被同学看到了，以及越来越暴露的危险的朋友圈。 

关于此事同样有在朋友圈中以及一个话唠社区中写道，后续似乎也会撰写一篇文章。

后话

其实在这件事情上就可以看得出我自己对个人的热点其实是有拖延行为的，

如果执行力很强的话，就当即发布文章，当即在博客内进行更新。

最近的生活都不是那么的好，这方面的发布的文章也很躁。

这段时间什么时候才能过去呢，或许是毕业之后？
