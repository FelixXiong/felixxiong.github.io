做了运营编辑之后，我发现了我的几点转变（一）

发表于 2018-09-23 | 分类于 运营 | |
字数统计: 2k | 阅读时长 ≈ 6

这是我做了运营之后的改变，最显著的改变是我不得不去看那些“从众的”“大众化的”“可能是鸡毛蒜皮的小事、大都是茶余饭后的八卦“和明显的出轨恋情，因为我认为他们的生活对我的发展并没有任何帮助。

记得崇旭老师写过一篇文章，大意是说，”如果吸引你注意力的事你不能改变它的走向，就尽快逃出来“。

我对娱乐新闻以及明星真的无感。

音乐以及视频应用（包括YouTube）的热点推送，我都是关闭的。

可是做了运营，人们却都**的喜欢，尤其是那些年轻人。

这是我的改变第一篇，第二篇于近期发出，欢迎关注。

还记得我之前在知乎上的一条回答：

大学可以忙些什么？

我在学一种软件的时候，老师给我们布置作业的时候我所做的事情就是（提前）下颏和这个专业相关的app，例如我这几周学的是Premirer，手机里就下载了和视频相关的一切app，如下图。

手机上则下载Art Station，国外知名的CG和插画等分享网站。

在这里我还放上了一幅图，是我用iPhone6s的时候的图。

在学校的时候我确实是这样，在做专门的事情的时候就喜欢下载专门的软件应用以及App，独自一人坐在学校图书馆的门前，开始带着我的beats耳机进行发掘。

在学习视频的时候，我便会下载YouTube，Vimeo进行学习，在需要创作广告的时候，我当时使用的是iPhone6s，还清晰的记得我下载了Launchpad应用进行音乐创作，当然鼓点很大，当时还觉得或许会对心脏没什么好处。

其实这样里应该有之前我一个人在创作音乐时的截屏和照片，后续筛选出当时的照片的时候再添加吧。

后续自己做一些事的时候叫别人拍几个照片。

进入运营，采集内容

现在开始买入运营的行业了，在入职第一天配置我的Mac（也就是Mac Chrome）的油猴脚本的时候，的一些转变：

  1. 开始不取消查看热门微博以及热门视频；
  2. 开始保留微博的公告栏；
  3. 开始关注热搜榜；
  4. 开始发现微博热搜的一些变化以及勇敢的在朋友圈发送出来;
  5. 在微信公众号觉得很好的地方，便会积极节选发到朋友圈，即使这样的文章似乎很快就会被相关原因进行屏蔽。

在此放上一张自己目前的配置的几张截图，欢迎与我交流。红框处为原本我会屏蔽的内容。

微博实在是太乱了。

阅读以及来源，保存

当然，最后一点必须拥有一个成熟的保存文章的体系，在此需要感谢好奇心日报、少数派以及AppSo等科技网站的勇敢科普以及宣传，当然还有支撑着这些网站的朋友们。

作为运营，多读书是固然重要的一件事，这一点我也很了解。可是在现在的“碎片化阅读当道”的时代下，随手摘录成了很重要的一个话题。

我在之前的公众号里也学过我的一个自己的阅读体系：详情可以查看这篇文章：

RSS，关注更多的订阅源进行合理的分类，可以省下手机不少空间，之前我上面的文章中也提到过。

关注推送

因为各种应用的推送，之前都觉得是垃圾，因为自己在做公众号象限的小屋的时候确实就是不喜欢跟热点的，图片是自己制作，灵感来源于生活，非常的自由。

可是当变成了工作，得抓住手机的每一个应用软件的推送，所以在这里，网易云音乐的音乐歌单，微博的各个推送，都要关注，因为他们都有可能成为（微信）公众号文章的内容之一。

以及仍旧不看头条和抖音。[2]

现在仍然没有下载，防止和吃鸡一样上瘾。

关于转变以及工作上的迎合以及优劣

说着说着，似乎如果把工作带入到了生活，似乎可真的算是认真了，抓住某一个热点，可是你也真的抓不住热点。

因为不是所有的热点你都知道，也不是所有的热点你都能写，即使因为某些总所周知的原因会从热搜里消失，会从微信公众平台里屏蔽，还有真的是因为一些事情，还有名誉。

把工作带入生活，会将自己改变么？

以下是优势。

似乎会对同龄人来说更有谈资，例如之前真的一向不是很喜欢综艺与娱乐节目，可是做了运营之后，或许会和同龄人有了共同话题，或许变得更加的投机，或许……

这就是社交中的能力之一了。

前些日子看到朋友的一篇文章说，“充满性暗示的……”我当时不解，在底下评论留言问，为什么会牵扯到性？

他回复我：

一切行为皆为性行为。具体的文章，可以看看弗洛伊德的著作。

由此得知，做了运营，不得不去看一些自己原本很奇怪的问题，就例如做是为什么会牵扯到“性”这件事上。

还有，自己之前买的书：《乌合之众：大众心理研究》。

下面说说劣势。

做了运营，每天的自己的信息流筛选体系就变得更加不起作用：你必须下载一些东西，看原本不喜欢看的东西，或是没有兴趣的东西，甚至是调整自己说话的姿态：根据目标人群定位，说一些大众可以看懂的话，说一些大众必须感兴趣的话。

就是这样。

  1. 自从自己制作了一个Xcode报错集合以及Android报错集合后，开始觉得自己比较良心的一点就是直接在内容当中插入相应的语句，而不是顺手只放一个链接，让你去看文章然后顺便增加一个阅读量，推广那是以后的事了，因为觉得推广的话真的好才可以，也是后话了。 ↩︎

  2. 说到抖音，之前在北京面试某A轮的天气软禁的时候和产品聊到抖音，我当时没玩，自称自己是不是老了。现在还是觉得这真的是个浪费时间的东西。 ↩︎
